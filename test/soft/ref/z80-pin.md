DoutはData信号のZ80からの出力信号が出ているもの
Z80からのDataを読み込む時にDoutを使うかDを使うかはオプションで切り替えられるようにすること

| 種類         | io     | とりうる値 | デフォルト値 | 名前     |
|--------------|--------|------------|--------------|----------|
| アドレスバス | out    | 1/0/Z      | Z            | A0~A15   |
| データバス   | in/out | 1/0/Z      | Z            | D0~D7    |
| バス制御     | in     |            |              | /BUSRQ   |
|              | out    | 1/0        | 1            | BUSAK    |
| 主記憶制御   | out    | 1/0/Z      | 1            | /MREQ    |
|              | out    | 1/0/Z      | 1            | /RD      |
|              | out    | 1/0/Z      | 1            | /WR      |
|              | out    | 1/0        | 1            | /RFSH    |
| 入出力制御   | out    | 1/0/Z      | 1            | /IORQ    |
| その他の制御 | out    | 1/0        | 1            | /M1      |
|              | in     |            |              | /RESET   |
|              | in     |            |              | /WAIT    |
|              | out    | 1/0        | 1            | /HALT    |
| 割り込み入力 | in     |            |              | /NMI     |
|              | in     |            |              | /INT     |
| クロック     | in     |            |              | /CLK     |
| 電源         | in     |            |              | VCC(5V)  |
|              | in     |            |              | GND      |
